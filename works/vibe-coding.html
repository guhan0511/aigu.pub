简历一页纸？不如动态个人主页

Personal Homepage Project

项目概述

简历是阶段性的静态展现，在这个成长极快的时代，简历更新的速度难免跟不上个人迭代的速度，因此本人第一个vibe coding作品决定手搓一个《个人主页》，用持续更新的个人履历和近期关注，来替代个人简历。用图+文的形式，留下直观的第一印象。

技术栈

HTML5 CSS3 JavaScript GSAP

打算做同款个人主页？注册域名、部署服务器遇到了问题？欢迎和阿一古讨论在哪里能注册到性价比最高的域名。也欢迎你分享更炫酷的个人主页。

02

Chat EXCEL

项目概述

想做趋势图、散点图，厌倦了学一套漂亮的python代码再复刻绘制，或是不想用数据分析软件里没有特色的图表模板。试着用AI coding, 上传你的.dta .excel数据，简单明了的表述你的需求，接下来就是去喝一杯咖啡，等着验收成果。

想监控各个城市的销售数据，业绩达标情况？Tableau里面想要有交互地图要么需要地图元件，要么需要按照格式精准地提供省份、城市名字。可地图元件数据并不是那么好找，一个一个对省份城市名字又费时费力。Chat EXCEL帮你按照Tableau的模板，自动把需要的字段填补，放到Tableau里，漂亮的地图交互面板就出来啦。这里我以印尼这个千岛之国为例，检验一下是否一个小岛都没有落下。
